1) JavaScript String
A string is a data type used to represent textual data. For example, "hello", "JavaScript", etc. are strings. 
It contains a sequence of characters.

In JavaScript, strings are surrounded by quotes:

Single quotes: 'Hello'
Double quotes: "Hello"   
Backticks: `Hello`  

// string enclosed within single quotes
let fruit = 'apple';
console.log(fruit)

// string enclosed within double quotes
let country = "USA";
console.log(country);

// string enclosed within backticks
let result = `fail`;
console.log(result);


Note : We can find data type by using typeof() function. 



2) JavaScript Number
In JavaScript, number is a data type that represents numeric values (both integers and floating-point numbers).

Integers - Numeric values without any decimal parts. Example: 3, -74, etc.
Floating-Point - Numeric values with decimal parts. Example: 3.15, -1.3, etc.

// integer value
let number1 = -3;
console.log(number1);

// floating-point value
let number2 = 3.15;
console.log(number2);



3) JavaScript Boolean
A Boolean data can only have one of two values: true or false. This data type represents logical entities.

let dataChecked = true;
console.log(dataChecked);  // true

let valueCounted = false;
console.log(valueCounted);  // false




4) JavaScript undefined
In JavaScript, undefined is a special value that represents the absence of a value or an uninitialized state.
If a variable is declared but the value is not assigned, then the value of that variable will be undefined.
It is also possible to explicitly assign undefined as a variable value.

let name;
console.log(name);  // undefined

let name = undefined;
console.log(name);  // undefined





5)JavaScript null
In JavaScript, null is a special value that represents "no value" or "nothing."

let number = null;
console.log(number);  // null
